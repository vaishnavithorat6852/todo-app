# 📝 Todo List App

A simple todo list app I built while learning HTML, CSS and JavaScript 🙂

🔗 **Live Demo** → [click here](https://vaishnavithorat6852.github.io/todo-app/)

## ✨ What it does

- ✅ Add, complete and delete tasks
- 🔍 Filter by All / Active / Completed
- 💾 Tasks don't disappear on refresh — thanks to localStorage
- ⌨️ Hit Enter to add a task quickly
- 📱 Works on mobile too


## 🛠️ What I used

- 🌐 HTML
- 🎨 CSS (Flexbox)
- ⚡ JavaScript (DOM Manipulation, localStorage)


## 🚀 Run it locally

1. Clone the repo
2. Open `index.html` in your browser
3. That's it — no installs, no setup 🙌
